Hello World in JavaScript


console.log('Hello World');

In JavaScript, console.log() is used to print any message or value to the console. In this example, we are printing Hello World to the console.


To run this code, open the browser console and paste the above code. You will see the output as Hello World.


You can also run this code in the JavaScript file. Create a new file with the .js extension and paste the above code. Now open the file in the browser, and you will see the output in the console.


This is the simplest way to print any message or value to the console in JavaScript.
